# Agent Skill Security Evaluator

サードパーティのAgent Skillsを8つのセキュリティドメインで多面的に評価し、構造化された脆弱性レポートを生成するClaude Codeプラグインです。

## 概要

Agent Skill Security Evaluatorは、第三者が開発・公開したAgent Skillsのセキュリティリスクを自動的に評価します。SKILL.mdファイル、GitHubリポジトリ、スクリプトファイル、またはインラインコードを入力として受け取り、18項目のチェックを10段階で評価し、総合スコア・リスク判定・改善提案を含む詳細なレポートを出力します。

## 評価ドメイン

| # | ドメイン | 重み | 内容 |
|---|----------|------|------|
| D1 | 配布元・信頼性 | 1.0x | 開発者の身元、配布チャネル、メンテナンス状況 |
| D2 | コード・静的解析 | 1.5x | SKILL.md指示の安全性、難読化検出、スクリプト安全性 |
| D3 | 依存関係・サプライチェーン | 1.2x | パッケージの安全性、MCP・外部サービス連携 |
| D4 | Prompt Injection | 1.5x | Direct/Indirect Prompt Injectionリスク |
| D5 | データ・プライバシー | 1.3x | 機密データ処理、個人情報保護 |
| D6 | 権限・アクセス制御 | 1.2x | ファイルシステム・OS権限、ネットワークアクセス |
| D7 | 運用・保守性 | 0.8x | ドキュメント、テスト・品質保証 |
| D8 | エージェント動作・永続性 | 1.4x | Hookメカニズム安全性、スキル自己増殖リスク |

## 使い方

### スキルの呼び出し

以下のようなキーワードで自動的に起動します：

- 「スキルを評価して」
- 「セキュリティチェックして」
- 「このスキルは信頼できる？」
- 「Agent Skillのリスク評価」

### 入力形式

- SKILL.mdファイル
- GitHubリポジトリURL
- スクリプトファイル（.py / .js / .sh等）
- 会話内に貼り付けられたコード

### 評価結果

10段階スコアリングによる判定：

| スコア範囲 | 判定 | アクション |
|-----------|------|-----------|
| 85-100 | 承認 | 通常使用を承認。四半期ごとに再評価 |
| 70-84 | 条件承認 | 監視付きで使用可 |
| 50-69 | 要改善 | High項目の解消まで使用保留 |
| 0-49 | 使用禁止 | 代替スキルを選定 |
| 自動失格 | 即時排除 | スコアに関わらず使用禁止 |

## インストール

```bash
claude plugin add --from <リポジトリURL>
```

## ディレクトリ構成

```
.claude-plugin/
  plugin.json          # プラグインメタデータ
  marketplace.json     # マーケットプレイス設定
skills/
  evaluator/
    SKILL.md           # 評価スキル本体
    references/
      evaluation-rubric.md     # 18項目の10段階評価基準
      scoring-aggregation.md   # スコア集計・判定ロジック
      report-template.md       # レポート出力テンプレート
```

## 主な機能

- **8ドメイン x 18項目**の包括的セキュリティ評価
- **10段階スコアリング**による定量的リスク判定
- **自動失格条件**による致命的リスクの即時検出
- **重み付きスコア集計**（D2/D4/D8は高重み）
- **複数リポジトリの同時評価**対応
- **構造化レポート**出力（エグゼクティブサマリー、改善提案付き）

## 注意事項

- 静的解析のみ実施。動的実行・サンドボックステストは含みません
- 評価結果はあくまで参考情報です。最終判断は組織のセキュリティポリシーに従ってください
- スコアは提供された情報に基づく推定値であり、完全な保証を意味しません
