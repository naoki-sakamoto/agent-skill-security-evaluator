## Step 3-2: 自動失格チェック（最優先・計算前に必ず実行）

**以下のいずれかに該当する場合、計算を行わずに即座に ⛔ を確定する：**

```
自動失格条件：
□ いずれかの項目のスコアが 1 または 2
□ D4（Prompt Injection）ドメインの平均スコアが 4 未満
□ D2-3（スクリプト安全性）のスコアが 3 以下
□ D5-1（機密データ処理）のスコアが 2 以下

□ D8-2（スキル自己生成）のスコアが 2 以下        ← v2.0追加
```

自動失格が確定した場合はレポート冒頭に表示し、残りの評価詳細は参考情報として提示する：

```
⛔ 自動失格：[失格条件の説明]
   当該スキルは詳細な審査なしに使用禁止とします。
   以下の評価詳細は参考情報として提示します。
```

## Step 3-3: ドメイン平均スコアの算出（8ドメイン分を明示）

```

D2平均 = (D2-1 + D2-2 + D2-3) ÷ 3
D3平均 = (D3-1 + D3-2) ÷ 2
D4平均 = (D4-1 + D4-2) ÷ 2
D5平均 = (D5-1 + D5-2) ÷ 2
D6平均 = (D6-1 + D6-2) ÷ 2

```

## Step 3-4: 加重スコアの算出（8ドメイン分を明示）

```
D1加重 = D1平均 × 1.0 × 3 = D1平均 × 3.0    （最大 30）
D2加重 = D2平均 × 1.5 × 3 = D2平均 × 4.5    （最大 45）
D3加重 = D3平均 × 1.2 × 2 = D3平均 × 2.4    （最大 24）
D4加重 = D4平均 × 1.5 × 2 = D4平均 × 3.0    （最大 30）
D5加重 = D5平均 × 1.3 × 2 = D5平均 × 2.6    （最大 26）
D6加重 = D6平均 × 1.2 × 2 = D6平均 × 2.4    （最大 24）
D7加重 = D7平均 × 0.8 × 2 = D7平均 × 1.6    （最大 16）
D8加重 = D8平均 × 1.4 × 2 = D8平均 × 2.8    （最大 28）← v2.0追加

加重合計 = D1加重+D2加重+D3加重+D4加重+D5加重+D6加重+D7加重+D8加重
最大加重合計 = 30+45+24+30+26+24+16+28 = 223    ← v2.0更新
```

**総合スコア（100点換算）** = Round((加重合計 ÷ 223) × 100, 1)

## Step 3-7: Critical/High項目の集計

```
Critical項目（スコア1-3）：
  → 項目コード、スコア、発見した証拠・行番号の要約

High項目（スコア4-5）：
  → 項目コード、スコア、懸念点の要約

改善優先度：
  Priority 1（即時）: Critical項目（スコア1-3）、自動失格条件該当項目
  Priority 2（早急）: High項目（スコア4-5）
  Priority 3（計画的）: 要注意項目（スコア6）
  Priority 4（任意）: 軽微な改善点（スコア7-8）
```

## 計算例（v2.0・D8含む18項目）

```
チェック項目スコア：
D1-1: 8, D1-2: 9, D1-3: 7  → D1平均 = 8.0
D2-1: 5, D2-2: 9, D2-3: 8  → D2平均 = 7.33
D3-1: 8, D3-2: 10           → D3平均 = 9.0
D4-1: 9, D4-2: 7            → D4平均 = 8.0
D5-1: 7, D5-2: 8            → D5平均 = 7.5
D6-1: 9, D6-2: 9            → D6平均 = 9.0
D7-1: 6, D7-2: 5            → D7平均 = 5.5
D8-1: 8, D8-2: 9            → D8平均 = 8.5  ← v2.0追加

自動失格チェック：最低スコア=5 → 失格なし

加重計算：
D1: 8.0  × 3.0 =  24.0
D2: 7.33 × 4.5 =  33.0
D3: 9.0  × 2.4 =  21.6
D4: 8.0  × 3.0 =  24.0
D5: 7.5  × 2.6 =  19.5
D6: 9.0  × 2.4 =  21.6
D7: 5.5  × 1.6 =   8.8
D8: 8.5  × 2.8 =  23.8  ← v2.0追加

加重合計 = 24.0+33.0+21.6+24.0+19.5+21.6+8.8+23.8 = 176.3
総合スコア = (176.3 ÷ 223) × 100 = 79.1点

ペナルティ：D2-1がスコア5でD2平均が6未満 → −5点
最終スコア = 79.1 − 5.0 = 74.1点

判定：🟡 条件承認
```
